A simple and elegant portfolio showcase plugin with Elementor support. Create and manage beautiful portfolio items using a custom post type and display them with Elementor widgets.

== Description ==

**Portfolio Showcase** helps you display portfolio items professionally and beautifully. It registers a custom post type for portfolio entries and provides Elementor widget integration so you can easily add the showcase anywhere on your site.

**Key Features:**

* Custom Post Type for portfolio items  
* Elementor Widget for displaying portfolio  
* Templates for single and archive views  
* Lightweight and clean code  
* SEO-friendly markup  
* Fully responsive design

This plugin is ideal for freelancers, designers, developers, and creative agencies looking to showcase their work.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/portfolio-showcase/` directory, or install it via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add portfolio items from the **Portfolio** menu in the dashboard.
4. Use the Elementor widget to display them on any page.

== Frequently Asked Questions ==

= Does it require Elementor Pro? =  
No. The plugin works with the free version of Elementor.

= Can I use my own template designs? =  
Yes. You can override the default templates by copying them into your theme.

= Will it slow down my site? =  
Not at all. The plugin is lightweight and optimized for performance.
